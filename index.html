Unit 1, Stage 0: Getting Started with HTML

Key concepts:
The Web/Internet – The world’s largest computer network
HTML - HyperText Markup Language – A system for tagging text files to achieve font, color, graphic, and hyperlink effects on Web pages.
URL – location of resources on The Internet (Web addresses)
HTTP – HyperText Transport Protocol - The protocol of Web pages
Web Servers – Computers on the Internet that host Web services/applications

World Wide Web
HTML is the main document type on the Web
major pieces of the Web : user/computer +browser <-->Internet <--> servers

HTML
Made up of:
- text content
- markup (what it looks like)
- references to other documents (e.g. images and videos)
- links to other Web pages

HTML Markup element components:
- opening tags
- contents
- closing tags
- void tag – Tag that has no content. It does not need a closing tag.

Common HTML tags:
- bold : b - makes the encapsulated content boldface
- emphasis : em - italicizes the encapsulated content
- anchor : a - provides a link to other Web resources.
- image : img - places an image in the Web page by pointing to the link of the image
- line break : br - adds a new line for text. This is a void tag (no closing tag)

Basic HTML vocabulary:
- Tag : HTML tags are always contained with angled brackets
- Element : an HTML element refers to everything within a set of opening and closing tags.
- Attribute : this is a property of an HTML tag. example: href=www.reddit.com can be the attribute of a anchor tag

In-line vs. Block elements
In-line elements take up only as much width as needed, and do not force new lines.
Block elements form an "invisible box" around the content inside of them.
Examples of in-line elements are span and br
Examples of block elements are p and div
